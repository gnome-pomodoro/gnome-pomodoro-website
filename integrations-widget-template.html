Connect any Time Tracking Software and create Time Entries.

Install

  1. sh -c "$(curl -fsSL https://raw.githubusercontent.com/gnome-pomodoro/gnome-pomodoro-tracking/master/startup.sh)" "" --install

Go to Gnome Pomodoro Preferences / Plugins ... / Custom Actions(Execute shell scripts) / Add

  1. gnome-pomodoro-tracking -gps "$(state)" -gpt "$(triggers)" -gpd "$(duration)" -gpe "$(elapsed)"

CLI

  1. gnome-pomodoro-tracking --plugin toggl
  2. gnome-pomodoro-tracking --name first-time-entry
  3. gnome-pomodoro-tracking --status
  4. gnome-pomodoro-tracking --restart
  5. gnome-pomodoro-tracking --stop
